1. Correlation Analysis

  • 매월 유튜브에 광고 비용을 지출하여 신규 유저 획득 연관성 파악
from scipy import stats
import numpy as np
import pandas as pd

marketing_costs = [352, 164, 210, 425, 503, 232, 321, 556, 464, 578, 612, 434] # 집행된 유튜브 광고 비용 (단위 : 10,000원)
user_acquired = [7214, 6122, 6896, 8020, 10982, 9021, 9240, 10210, 9987, 11521, 9792, 9852] # (단위 : 명)

ad_df = pd.DataFrame({'Marketing_Costs':marketing_costs, 'User_Acquired':user_acquired})
  • CAC(Customer Acquisition Cost, 신규고객 유치 비용) 계산
cac = ad_df['Marketing_Costs'].sum() / ad_df['User_Acquired'].sum()

print('1명의 신규 유저를 획득하기 위한 평균적인 유튜브 광고 비용 : {:.0f}원'.format(cac * 10000)) # .0f : 소수점 0번째 자리까지 출력

1명의 신규 유저를 획득하기 위한 평균적인 유튜브 광고 비용 : 446원

→ 정말 446원을 쓰면 1명의 신규 유저를 획득할 수 있을까?

→ 마케팅 비용과 신규 유저 획득 상관성 파악 필요

  • 상관분석
result = stats.pearsonr(ad_df['Marketing_Costs'], ad_df['User_Acquired']) # Pearson’s co"r"relation coefficient

print("피어슨 상관계수 :{:.4f}".format(result[0]))
print("p-value :{:.4f}".format(result[1]))

피어슨 상관계수 :0.8036 p-value :0.0016

→ p-value < 0.05이므로 유의수준 5% 하에서 유튜브 광고 비용과 신규 유저 획득 수는 상관관계가 있다고 판단할 수 있다. 상관계수는 0.8036으로 강한 양의 상관관계가 있다고 판단할 수 있다.

2. A/B Test with t-test

• 게임 A와 B를 플레이한 유저들이 실제로 이탈하기까지의 시간(체류시간, Duration time)을 측정

game_a = pd.DataFrame([20.5, 12.6, 19.5, 18.8, 13.4, 13.5, 17.5, np.nan, 12.8, 17.8, np.nan, 23.1, 10.6, np.nan, 11.5], 
                     columns=['Duration_A']) # np.nan : 결측치 생성
game_b = pd.DataFrame([11.8, 10.7, np.nan, 12.5, np.nan, 14.9, 12.1, 13.9, 10.3, 9.0, 13.3, 12.4, 12.5], 
                     columns=['Duration_B'])

print(game_a.mean())
print(game_b.mean())

Duration_A 15.966667

Duration_B 12.127273

  • 두 게임의 평균 체류시간의 차이가 정말로 있는 것일까? → t-test
# 결측치 삭제
game_a = game_a.dropna()
game_b = game_b.dropna()

# 독립표본 t-test (두 집단간의 평균차이를 검정)
stats.ttest_ind(game_a['Duration_A'], 
                game_b['Duration_B'], 
                equal_var=False) # 동분산 검정 결과 -> True : 동분산, False : 동분산 x

Ttest_indResult(statistic=3.0165632092150694, pvalue=0.008734970056646718)

→ p-value < 0.05이므로 유의수준 5% 하에서 게임 A와 게임 B의 평균 체류시간은 통계적으로 유의한 차이가 있다고 판단할 수 있다. 따라서 두 게임 중에 평균 체류시간이 더 높은 A를 선택할 것이다.

3. A/B Test with Chi-square test

  • 최종 구매를 위한 버튼(혹은 광고 배너)을 2개의 서로 다른 시안으로 제작해 각기 다른 유저들에게 노출하여 특정 지표가 더 높은 시안을 선택, 지표는 다음과 같은 지표를 고려
  • Conversion rate (전환율)
# 데이터가 빈도 수로 구성된 테이블일 경우로 가정
a_clicked = 144 # 버튼(배너) 시안 A를 누른 유저의 수
a_unclicked = 2362 # 버튼(배너) 시안 A를 보았으나 누르지 않은 유저의 수

b_clicked = 212 # 버튼(배너) 시안 B를 누른 유저의 수
b_unclicked = 2528 # 버튼(배너) 시안 B를 보았으나 누르지 않은 유저의 수

click_df = pd.DataFrame({'Clicked':[a_clicked, b_clicked], 
                         'Unclicked':[a_unclicked, b_unclicked]}, index=['Button_A', 'Button_B'])
click_df
ClickedUnclicked
Button_A1442362
Button_B2122528
# 단순한 전환율 계산
conversion_rate = click_df['Clicked'] / (click_df['Clicked'] + click_df['Unclicked']) * 100

print("Button(or Banner Ad) A's Conversion Rate(or CTR) is : {:0.2f}%".format(conversion_rate[0]))
print("Button(or Banner Ad) B's Conversion Rate(or CTR) is : {:0.2f}%".format(conversion_rate[1]))

Button(or Banner Ad) A's Conversion Rate(or CTR) is : 5.75%

Button(or Banner Ad) B's Conversion Rate(or CTR) is : 7.74%

  • 두 버튼의 클릭 빈도의 차이가 정말로 있는 것일까? → chi-square test
p_value = stats.chi2_contingency([click_df['Clicked'], click_df['Unclicked']])[1]
print('p-value = {:.4f}'.format(p_value))

p-value = 0.0050

→ p-value < 0.05이므로 유의수준 5% 하에서 버튼 A와 버튼 B의 클릭 수는 통계적으로 유의한 차이가 있다고 판단할 수 있다. 따라서 두 버튼 중에 클릭빈도가 더 높은 B를 선택할 것이다.

→ 하지만 클릭 수 자체가 미미하다면 검정 결과에도 불구하고 실질적인 효용이 적기 때문에 이를 고려하여 판단해야한다.
